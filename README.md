# TODO APIS

## Install

clone the repo and install dependencies:-

```bash
git clone https://github.com/Ritesh560/node-todo-api.git
cd <project-name>
npm install
```

## start the server

```bash
nodemon app.js
```

OR

```bash
node app.js
```

### The server will now live and you can visit it on http://localhost:8080/
